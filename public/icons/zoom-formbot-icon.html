üìù How to use:

  1. Adjust the Zoom Level slider to make the beaver bigger (2-3x works well)
  2. Use the position sliders to center the beaver perfectly
  3. Check the "Chrome Toolbar Preview" to see how it will look
  4. Click "Download Zoomed formbot_icon.png"
  5. Replace public/icons/formbot_icon.png with the downloaded file
  6. Run npm run build to rebuild

Tip: A zoom of 2.0-2.5x usually works best for toolbar visibility!
